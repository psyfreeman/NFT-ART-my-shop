# Cannabis Seed Catalog Website - MVP Todo

## Project Overview
Create a copy of Barneys Farm website for displaying cannabis strains with the ability to add custom strains.

## MVP Features
1. Modern landing page with hero section
2. Strain catalog/grid display
3. Individual strain detail pages
4. Responsive design
5. Search and filter functionality
6. Admin-like interface to add new strains
